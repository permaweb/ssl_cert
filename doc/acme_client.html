      View Source acme_client (ssl_cert v1.0.0)

      ACME client module for Let's Encrypt certificate management.

      This module provides the main API for ACME (Automatic Certificate Management Environment) v2 protocol operations. It serves as a facade that orchestrates calls to specialized modules for HTTP communication, cryptographic operations, CSR generation, and protocol implementation.

      The module supports both staging and production Let's Encrypt environments and provides comprehensive logging through HyperBEAM's event system.

      This refactored version delegates complex operations to specialized modules:

      • acme_protocol: Core ACME protocol operations
      • acme_http: HTTP client and communication
      • acme_crypto: Cryptographic operations and JWS
      • acme_csr: Certificate Signing Request generation
      • acme_url: URL parsing and manipulation utilities
